# ZyLord

ZyLord är en Discord client mod med Docker Compose-liknande installationssystem.

## Kom igång

1. Installera beroenden:
   ```bash
   npm install
   ```

2. Kör installationsprogrammet:
   ```bash
   npm start
   ```

## Användning

Redigera `zy-lord.yml` för att konfigurera plugins och inställningar.

### Kommandon

- `up` - Installera ZyLord
- `down` - Avinstallera ZyLord
- `build` - Bygg om
- `ps` - Visa status
- `pull` - Uppdatera plugins
- `logs` - Visa loggar
